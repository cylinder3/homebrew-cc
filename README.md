## Install

```bash
brew tap cylinder3/cc
brew install naiveproxy1
```

## Update

```bash
brew update
brew upgrade naiveproxy1
```

## Uninstall

```bash
brew uninstall naiveproxy1
brew untap cylinder3/cc
```

## Usage

```bash
brew services start  naiveproxy1
```
