## Install

```bash
brew tap cylinder3/cc
brew install xray1
```

## Update

```bash
brew update
brew upgrade xray1
```

## Uninstall

```bash
brew uninstall xray1
brew untap cylinder3/cc
```

## Usage

```bash
brew services list
brew services start xray1
brew services restart xray1
```
